Criar um Novo Orçamento

Para começar a usar o sistema, você precisa criar um orçamento:

  1. Clique no botão "Novo Orçamento" no topo da página
  2. Um novo orçamento será criado automaticamente
  3. Você pode gerenciar múltiplos orçamentos usando o menu lateral

Registrar Entradas

Para registrar suas receitas e entradas:

  1. Na seção "Entradas Mensais", clique em "Adicionar Categoria"
  2. Digite o nome da categoria (ex: Salário, Freelance)
  3. Preencha os valores para cada mês
  4. Os totais são calculados automaticamente

Registrar Gastos

Para registrar seus gastos e despesas:

  1. Na seção "Gastos Mensais", clique em "Adicionar Categoria"
  2. Digite o nome da categoria (ex: Alimentação, Transporte)
  3. Preencha os valores para cada mês
  4. Os totais são atualizados automaticamente

Contas Fixas

Para registrar suas contas fixas mensais:

  1. Na seção "Contas Fixas", clique em "Adicionar Categoria"
  2. Digite o nome da conta fixa (ex: Aluguel, Internet)
  3. Preencha os valores para cada mês
  4. Os valores são somados automaticamente aos gastos

Gerar Relatórios

Para gerar relatórios em PDF:

  1. Clique no botão "Gerar Relatório PDF"
  2. O sistema irá criar um relatório completo
  3. O relatório inclui gráficos, tabelas e resumo financeiro
  4. O arquivo será baixado automaticamente
